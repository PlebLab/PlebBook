Purpose: Capture presentation and UX constraints for PlebBook as a documentation site.

# Design Rules

## Visual Direction
- Preserve clean documentation readability over decorative styling.
- Keep typography and spacing stable across desktop and mobile.
- Avoid layout shifts and overlapping content sections.

## Interaction
- Sidebar navigation and page links must remain reliable.
- Embedded links should degrade gracefully when custom embed blocks are converted.
- Keep load behavior lightweight and fast.

## Accessibility Baseline
- Maintain semantic heading structure in markdown.
- Ensure color contrast remains readable in default theme.
- Prefer explicit link text where possible.
